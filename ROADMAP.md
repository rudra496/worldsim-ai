# Roadmap

## v0.1 — Current (March 2026)
- [x] Core simulation engine with formal state model
- [x] Agent-based modeling (4 agent types)
- [x] Grid and graph world environments
- [x] Zone system (residential, industrial, commercial, etc.)
- [x] Resource management
- [x] AI prediction and anomaly detection
- [x] LP-based optimization (resource allocation)
- [x] Scenario engine with 4 demo scenarios
- [x] FastAPI REST + WebSocket API
- [x] React 2D visualization dashboard
- [x] Docker Compose setup
- [x] Config-driven architecture
- [x] Metrics collection and export
- [x] Deterministic mode for reproducibility

## v0.2 — AI Enhanced
- [ ] PyTorch-based prediction models (LSTM, Transformer)
- [ ] Reinforcement learning agents
- [ ] Multi-agent AI system (Planner, Predictor, Optimizer)
- [ ] Feedback loop: simulation → AI → optimization → updated sim
- [ ] Advanced anomaly detection (isolation forest, autoencoder)

## v0.3 — 3D Visualization
- [ ] Three.js/WebGL 3D world view
- [ ] Camera controls (orbit, zoom, follow agent)
- [ ] 3D building and terrain rendering
- [ ] Agent path visualization in 3D

## v0.4 — Real-Time Data
- [ ] IoT sensor data ingestion
- [ ] MQTT/Kafka integration
- [ ] Live data streaming to simulation
- [ ] Real-time anomaly alerting

- [ ] Distributed state synchronization
